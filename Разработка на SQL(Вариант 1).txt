Таблица Courses - Курсы:

ID курса (CourseID) - первичный ключ
Название курса (CourseName)
Описание курса (CourseDescription)
Количество мест на курсе (CourseCapacity)

Таблица Schedule - Расписание:

ID записи (ScheduleID) - первичный ключ
ID курса (CourseID) - внешний ключ, связанный с таблицей "Курсы"
Дата начала курса (StartDate)
Дата окончания курса (EndDate)

Таблица Trainees - Обучаемые:

ID обучаемого (TraineeID) - первичный ключ
Имя (FirstName)
Фамилия (LastName)

Таблица CourseAttendance - Участие в конкурсе:

ID записи (AttendanceID) - первичный ключ
ID обучаемого (TraineeID) - внешний ключ, связанный с таблицей "Обучаемые" (Должен быть уникальным ключом)
ID курса (CourseID) - внешний ключ, связанный с таблицей "Курсы" (Должен быть уникальным ключом)
Дата начала участия (StartDate)
Дата окончания участия (EndDate)

Связи между таблицами:  
1. Курсы и Расписание - один ко многим
2. Курсы и Участие в курсе - один ко многим
3. Обучаемые и Участие в курсе - один ко многим
